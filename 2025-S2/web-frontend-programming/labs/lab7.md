# Лабораторная работа №7 - Работа с React (Next.js)

## Задание

Освоить основы работы с React в среде Next.js: научиться создавать компоненты, управлять состоянием, обрабатывать пользовательские события и реализовывать интерактивную логику интерфейса с использованием Tailwind CSS

## Условия

* Использовать **Next.js**
* Использовать **React (JSX, JavaScript)** (не TSX, TypeScript)
* Использовать **Tailwind CSS**
* Все задания должны находиться **на одной странице**
* Каждое задание должно быть реализовано **как отдельный React-компонент**

## Структура проекта

Проект создается с помощью Next.js:

```bash
npx create-next-app@latest lab7-react
cd lab7-react
npm run dev
```

В процессе создания проекта необходимо включить Tailwind CSS

Основной файл страницы:

```txt
app/page.jsx
```

Компоненты могут быть:

* в одном файле (`page.jsx`)
* либо вынесены в папку `app/components/`

## Задания

### 1. Кнопка "Подписаться" - 30%

Реализовать интерактивную кнопку подписки с счетчиком подписчиков

#### Требования

1. На странице должна быть кнопка "Подписаться" и число подписчиков рядом с ней
2. Начальное количество подписчиков генерируется случайным образом при загрузке страницы (через `Math.random()`)
3. При первом нажатии на кнопку:

   * количество подписчиков увеличивается на 1
   * кнопка меняет цвет (например, становится зеленой)
   * текст кнопки может измениться (например, "Вы подписаны")
4. При повторном нажатии:

   * количество подписчиков уменьшается на 1
   * кнопка возвращает исходный цвет
   * состояние подписки отменяется
5. Поведение реализовать через состояние React (`useState`)
6. Оформление выполнить с помощью Tailwind CSS

### 2. Выбор числа от 1 до 9 - 35%

Реализовать компонент выбора числа с кнопками и стрелками

#### Требования

1. Отобразить 9 кнопок с числами от 1 до 9.
2. Отобразить две дополнительные кнопки:

   * стрелка "влево" (уменьшить число)
   * стрелка "вправо" (увеличить число)
3. В каждый момент времени выбрано только одно число
4. Выбранное число должно визуально отличаться:

   * другой цвет
   * рамка
   * либо подсветка
5. Поведение:

   * клик по числовой кнопке выбирает соответствующее число
   * стрелки переключают число на предыдущее или следующее
   * число не может быть меньше 1 и больше 9
6. Текущее выбранное число должно отображаться явно
7. Вся логика реализуется через одно состояние в компоненте
8. Оформление выполнить с помощью Tailwind CSS

### 3. Игра "Угадай число" - 35%

Реализовать простую игру угадывания числа

#### Требования

1. При загрузке компонента загадывается случайное число от 0 до 100
2. На странице должны быть:

   * текстовое поле ввода числа
   * кнопка "Проверить"
   * текстовое сообщение с результатом
3. Пользователь вводит число и нажимает кнопку
4. После нажатия:

   * если введенное число меньше загаданного - вывести сообщение
     "НЕ УГАДАЛИ, ЧИСЛО МЕНЬШЕ";
   * если введенное число больше загаданного - вывести сообщение
     "НЕ УГАДАЛИ, ЧИСЛО БОЛЬШЕ";
   * если число угадано - вывести сообщение о победе
5. Проверку ввода выполнять в JavaScript
6. Использовать управляемый input (`value`, `onChange`)
7. Компонент должен работать без перезагрузки страницы
8. Оформление выполнить с помощью Tailwind CSS
