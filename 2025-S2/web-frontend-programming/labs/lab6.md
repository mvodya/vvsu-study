# Лабораторная работа №6 - Основы JavaScript

## Задание

Освоить базовые возможности JavaScript: переменные, условия, функции, массивы, циклы, работу с DOM и простые взаимодействия со страницей. Научиться запускать код в Node.js и подключать скрипты в HTML

Работа состоит из **5 независимых частей**, каждая оценивается отдельно

### Задание 1. Базовые конструкции JavaScript (Node.js) - 20 %

Создать файл `task1.js` и выполнить следующее:

1. Объявить переменные (`let`, `const`) разных типов: число, строка, boolean
2. Написать условие `if...else`, которое:

   * сравнивает число с 100
   * выводит в консоль одну из двух строк: `"Число больше 100"` или `"Число меньше или равно 100"`
3. Создать массив из 5 элементов и вывести третий элемент
4. Сделать цикл `for`, который выводит все элементы массива
5. Вызвать `node task1.js` и убедиться, что вывод корректный

### Задание 2. Простая программа на Node.js - 20 %

Создать файл `task2.js` - небольшую мини-программу

Требования:

1. Реализовать функцию `rollDice()`, которая:

   * возвращает случайное число от 1 до 6 (имитация кубика)
2. Реализовать функцию `generateName()`, которая:

   * собирает строку из трех случайных слов из массива, например:

     ```js
     ["fire", "shadow", "wolf", "neon", "crystal", "storm"]
     ```

   * результат: `"neon-wolf-fire"`
3. В главном коде:

   * бросить "кубик" 3 раза,
   * сгенерировать 1 случайное имя
   * вывести все это в консоль в виде:

     ```bash
     Кубики: 2, 5, 1
     Имя: neon-wolf-fire
     ```

Файл запускается через:

```bash
node task2.js
```

### Задание 3. Первая HTML-страница с JavaScript - 20 %

Создать `task3.html` и подключить скрипт **через внешний файл** `task3.js`

Требования:

1. На странице должны быть:

   * заголовок `<h1>`
   * абзац `<p>`
   * кнопка `<button>Нажми меня</button>`
2. В `task3.js`:

   * выбрать кнопку через `document.querySelector`
   * повесить событие `click`
   * при нажатии менять текст `<p>` на `"Кнопка нажата!"`
3. Скрипт подключить **в конце `<body>`**:

  ```html
  <script src="task3.js"></script>
  ```

### Задание 4. Интерактивная форма - 20 %

Создать `task4.html` и подключить `task4.js`

Элементы на странице:

* текстовое поле `<input>`
* кнопка `"Добавить"`
* список `<ul></ul>` (изначально пустой)

Поведение:

1. При вводе текста и нажатии на кнопку:

   * текст из поля должен добавляться как новый `<li>` в список
2. Если поле пустое - ничего не добавлять
3. После добавления очищать поле
4. Реализовать все с помощью DOM-операций:

   * `querySelector`
   * `createElement`
   * `.value`
   * `.appendChild`

Это по сути простой "мини-todo" без удаления

### Задание 5. Страница с Tailwind + JavaScript - 20 %

Создать проект по структуре предыдущей лабораторной:

```txt
src/app.css
dist/output.css
task5.html
task5.js
```

В `app.css`:

```css
@import "tailwindcss";
```

Сборка:

```bash
npx @tailwindcss/cli -i ./src/app.css -o ./dist/output.css --watch
```

Страница `task5.html` (любой дизайн):

Создать небольшой интерактивный блок из HTML + Tailwind + JavaScript

**Обязательные элементы:**

1. Карточка (`div`) с Tailwind-оформлением:

   * фон, тень, рамка, отступы.
2. Заголовок, абзац.
3. Кнопка `"Сменить тему"`.

Поведение JavaScript:

1. Клик по кнопке добавляет/удаляет класс Tailwind:

   ```js
   element.classList.toggle("bg-gray-900");
   element.classList.toggle("text-white");
   ```

2. Тем самым карточка меняет светлую/темную тему.
3. Все должно работать без перезагрузки страницы.
