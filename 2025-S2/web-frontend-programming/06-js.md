# Основы Java Script

**JavaScript (JS)** - это язык программирования, который добавляет веб-страницам **логику и интерактивность**. Если HTML отвечает за структуру, а CSS - за внешний вид, то JavaScript делает страницу "живой": позволяет реагировать на клики, ввод текста, анимации, обновление данных и многое другое

## Где работает JavaScript

JS запускается в двух основных средах:

1. **В браузере** - управление страницей, изменение DOM, работа с формами, кнопками и т.д.
2. **В Node.js** - выполнение JavaScript вне браузера (на компьютере или сервере), что позволяет писать полноценные программные модули, утилиты и серверную логику

### ECMAScript

JavaScript основан на стандарте **ECMAScript (ES)** - это официальный набор правил языка

Каждая версия (ES5, ES6, ES2020 и т.д.) добавляет новые возможности

### Для чего используется

С помощью JS можно:

* Изменять содержимое страницы
* Реагировать на действия пользователя
* Отправлять запросы на сервер
* Создавать динамические интерфейсы
* Писать программы в Node.js

## Подготовка окружения и способы запуска JavaScript

### REPL в Node.js

Node.js имеет встроенную интерактивную консоль (REPL - *Read-Eval-Print Loop*). Она позволяет писать код построчно и сразу видеть результат

Запуск:

```bash
node
```

Либо приложение "Node.js"

Пример:

```js
> 2 + 3
5
```

### Запуск скрипта как файла

Можно создать файл, например `script.js`, и выполнить его через Node.js (в консоли):

```bash
node script.js
```

Обычно это используют для работы с логикой, вычислениями или отладкой небольших частей кода

### Скрипт прямо внутри HTML

JavaScript можно написать прямо в теге `<script>`:

```html
<script>
  console.log("Привет с HTML!");
</script>
```

Код выполняется сразу при загрузке страницы браузером

### Подключение внешнего файла

Предпочтительный способ - вынести JS в отдельный файл:

```html
<script src="app.js"></script>
```

Так код легче читать, обновлять и переиспользовать

## Основы синтаксиса

JavaScript использует простой синтаксис, включающий переменные, типы данных и базовые операции. Эти элементы используются во всем дальнейшем коде

### Комментарии

```js
// Однострочный комментарий

/*
Многострочный
комментарий
*/
```

Комментарии не выполняются и служат для пояснений в коде

### Переменные: `let` и `const`

```js
let value = 10;   // значение можно изменить
const pi = 3.14;  // значение остается постоянным
```

Для объявления переменных используются `let` и `const`

`var` в современном JavaScript не применяется

### Типы данных

JavaScript имеет несколько основных типов:

```js
let age = 18;           // number
let name = "Miku";      // string
let active = true;      // boolean
let user = { id: 1 };   // object
let list = [1, 2, 3];   // array
let unknown;            // undefined
```

### Арифметические операции

```js
let x = 10 + 5;
let y = x * 2;
let z = y % 3;    // остаток
let p = 2 ** 3;   // степень
```

Поддерживаются базовые математические операторы: `+`, `-`, `*`, `/`, `%`, `**`

### Вывод значения

```js
console.log("Результат:", x);
```

`console.log` используется для вывода информации в консоль - как в Node.js, так и в браузере

## Работа со строками и числами

### Строки

```js
let name = "Марк";
let message = "Привет, " + name;
```

Строки можно объединять с помощью оператора `+`

#### Шаблонные строки

```js
let age = 20;
let text = `Возраст: ${age}`;
```

Используют обратные кавычки и позволяют вставлять значения напрямую

#### Полезные методы

```js
"Hello".toLowerCase();   // "hello"
"Test".toUpperCase();    // "TEST"
"123".length;            // 3
```

Методы позволяют изменять регистр и получать длину строки

### Числа

JavaScript не разделяет целые и дробные числа - все это `number`

```js
let a = 10.5;
let b = Math.round(a);   // 11
let c = Math.floor(3.9); // 3
let d = Math.ceil(3.1);  // 4
```

#### Случайное число

```js
let random = Math.random();      // число от 0 до 1
let randomInt = Math.floor(Math.random() * 10); // от 0 до 9
```

## Логические конструкции и условия

Условия позволяют выполнять разные действия в зависимости от ситуации

### Логические значения

```js
true
false
```

### Операторы сравнения

```js
5 > 3      // больше
5 >= 3     // больше или равно
5 == "5"   // сравнение по значению
5 === 5    // строгое сравнение (по значению и типу)
```

В современном коде используется **строгое сравнение** `===` и `!==`

### Логические операторы

```js
true && false   // и
true || false   // или
!true           // отрицание
```

### Условный оператор if

```js
let age = 18;

if (age >= 18) {
  console.log("Доступ разрешен");
} else {
  console.log("Доступ запрещен");
}
```

### Несколько условий

```js
if (score > 90) {
  console.log("A");
} else if (score > 70) {
  console.log("B");
} else {
  console.log("C");
}
```

### Тернарный оператор

Короткая форма условия:

```js
let status = age >= 18 ? "adult" : "child";
```

## Циклы и повторение действий

Циклы позволяют выполнять одно и то же действие несколько раз

### Цикл for

Цикл с заранее известным количеством повторений

```js
for (let i = 0; i < 5; i++) {
  console.log("Шаг:", i);
}
```

### Цикл while

Используется, когда количество повторений заранее неизвестно

```js
let x = 3;

while (x > 0) {
  console.log(x);
  x--;
}
```

### Перебор массива: for...of

Удобный способ пройтись по элементам массива

```js
let numbers = [10, 20, 30];

for (let n of numbers) {
  console.log(n);
}
```

## Массивы и объекты

Массивы и объекты - основные структуры данных в JavaScript

### Массивы

Массив - упорядоченный список значений

```js
let numbers = [10, 20, 30];
```

#### Доступ к элементам

```js
numbers[0]; // 10
numbers[1]; // 20
```

#### Изменение массива

```js
numbers.push(40);  // добавить в конец
numbers.pop();     // удалить из конца
numbers.shift();   // удалить из начала
numbers.unshift(5); // добавить в начало
```

### Объекты

Объект хранит данные в виде пар "ключ - значение"

```js
let user = {
  name: "Anna",
  age: 19
};
```

#### Доступ к свойствам

```js
user.name;   // "Anna"
user.age;    // 19
```

#### Изменение свойств

```js
user.age = 20;
user.city = "Moscow";
```

Объект можно расширять новыми полями в любое время

## Функции

Функции позволяют объединять повторяющийся код и вызывать его по имени. Это основной способ структурировать логику

### Объявление функции

```js
function greet() {
  console.log("Привет!");
}
```

Вызов:

```js
greet();
```

### Параметры и аргументы

Функция может принимать данные:

```js
function add(a, b) {
  return a + b;
}

let sum = add(3, 5); // 8
```

`return` возвращает результат, который можно сохранить или вывести

### Стрелочные функции

Более компактная форма записи:

```js
const multiply = (x, y) => x * y;

let result = multiply(2, 4); // 8
```

Если тело состоит из одного выражения, фигурные скобки и `return` можно не писать

## Введение в DOM

DOM (Document Object Model) - это структура веб-страницы, которую браузер создает из HTML

JavaScript может получать доступ к элементам этой структуры и изменять их

### Поиск элементов

Элемент можно выбрать по id, селектору или классу

```js
document.getElementById("title");
document.querySelector(".card");
document.querySelector("#menu");
```

`querySelector` поддерживает любые CSS-селекторы

### Изменение текста и содержимого

```js
let title = document.getElementById("title");

title.textContent = "Новый текст";
title.innerHTML = "<b>Жирный текст</b>";
```

* `textContent` - безопасная замена текста
* `innerHTML` - вставка HTML (используется аккуратно)

### Изменение стилей

```js
let box = document.querySelector(".box");

box.style.backgroundColor = "lightblue";
box.style.padding = "20px";
```

Можно изменять любые CSS-свойства

### Классы

```js
box.classList.add("active");
box.classList.remove("hidden");
box.classList.toggle("open");
```

Удобно для переключения состояний интерфейса

### Создание и удаление элементов

```js
let item = document.createElement("div");
item.textContent = "Новый элемент";

document.body.appendChild(item);
```

Удаление:

```js
item.remove();
```

## События

События позволяют странице реагировать на действия пользователя: клики, ввод текста, прокрутку, отправку форм и т.д.

### Назначение обработчика события

Основной способ - `addEventListener`:

```js
let button = document.querySelector("#btn");

button.addEventListener("click", () => {
  console.log("Кнопка нажата");
});
```

### Событие ввода текста

```js
let input = document.querySelector("#name");

input.addEventListener("input", () => {
  console.log("Текущий текст:", input.value);
});
```

### Событие отправки формы

```js
let form = document.querySelector("#form");

form.addEventListener("submit", (e) => {
  e.preventDefault(); // отмена стандартной отправки
  console.log("Форма отправлена");
});
```

### Работа с событием

Обработчик может получать объект события:

```js
button.addEventListener("click", (event) => {
  console.log("Тип события:", event.type);
});
```
